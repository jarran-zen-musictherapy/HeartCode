# HeartCode

HeartCode is a browser-based audio processing tool designed to take digital stethoscope recordings and transform them into analyzed, processed, and rhythmically quantized audio for clinical music creation.

## Tech Stack
* **Framework:** React 18+ (via Vite)
* **Audio Engine:** Native Web Audio API
* **Visuals:** HTML5 Canvas (2D Context)

## Development Phases
* **Phase 1 (Current):** Core Engine MVP (File Loader, Waveform Canvas, Playback Transport, Spectrum Display).
* **Phase 2:** Analysis (Transient detection, BPM estimation).
* **Phase 3:** Processing (Parametric EQ, Compressor).
* **Phase 4:** Grid Warp (SoundTouchJS time-stretching).
* **Phase 5:** Export (WAV encoding, slicing one-shots).
